## 🚀 クイックスタート

### 新しいスクレイパーを実装する

1. [status.md](./status.md) で未実装ASPを確認
2. [guide.md](./guide.md) の手順に従って実装
3. [testing.md](./testing.md) で検証

### 既存スクレイパーをテストする

1. [testing.md](./testing.md) を開く
2. テストスクリプトを実行: `pnpm exec tsx src/scripts/test-scraper-data.ts`
3. 結果を確認

## ⚠️ 重要な仕様

### データ保存ルール

- **日次データ**: `daily_actuals` テーブルに保存（実際の日付）
- **月次データ**: `actuals` テーブルに保存（**月末日付**）

### データ分離ルール

**日次と月次を絶対に合算しないこと**

これはシステム全体の整合性に関わる最重要ルールです。
